Detailed Description

Classe utilitaire permettant d'interagir avec le canvas de l'application.

Donne accès à toutes les fonctions d'affichage sur la surface graphique de l'application. Trois accesseurs donnent accès aux informations de la surface :

  • largeur
  • hauteur
  • possède le focus Il existe plusieurs fonctions de dessin :
  • point
  • ligne
  • cercle
  • rectangle
  • polygone régulier
  • polygone convexe
  • image
  • texte
